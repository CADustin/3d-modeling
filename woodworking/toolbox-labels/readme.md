# Tool Box Label Generator

## Introduction

I started this project because I wanted to get more organized. I was inspired by some toolbox I remember seeing years ago where they had labeled drawers in a playful way (Smashy Things, Nuckle Busters, etc), and at the time I remember thinking it was the funniest thing. This project has been my take on that idea, and of course some serious labels too!
